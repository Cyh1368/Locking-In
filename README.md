# Locking-In: AI Attention Assistant

An AI agent that reduces attention loss from habitual app checking by providing context-aware nudges and soft blocking.

## Overview

This project implements an AI assistant that tracks app usage and context (calendar, tasks, unread counts) to decide whether to allow, nudge, or softly block app access.

## Development Setup

1. Ensure Python 3.8+ is installed.
2. Clone the repo: `git clone https://github.com/Cyh1368/Locking-In.git`
3. Navigate to the repo: `cd Locking-In`
4. Install dependencies: `pip install -r requirements.txt` (if any)
5. Run the visual demo: `python demo/visual_demo.py`

## Project Structure

- `src/`: Core source code
- `demo/`: Demo scripts and examples
- `docs/`: Documentation

## Running the Visual Demo

The visual demo simulates a computer desktop with app icons. Click on apps to interact, and the AI agent will show notifications.

To run: `python demo/visual_demo.py`
